# Aizume
MediaPipeを使用したウェブベースの手トラッキングアプリケーションです。

## デモ
[https://code4fukui.github.io/aizume/](https://code4fukui.github.io/aizume/)

## 機能
- MediaPipeを使用した手のトラッキング
- 複数の手のトラッキングに対応
- ミラーモード
- 背面カメラモード

## 使い方
1. [デモページ](https://code4fukui.github.io/aizume/)をウェブブラウザで開きます。
2. プロンプトに従ってカメラへのアクセスを許可します。
3. アプリケーションが検出した手の形状にランドマークが表示されます。
4. チェックボックスを使ってミラーモードと背面カメラモードを切り替えられます。
